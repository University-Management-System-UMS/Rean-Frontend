## create .env-fastlane for fastlane environment in `.env-fastlane`

```bash
    FASTLANE_APPLE_APPLICATION_SPECIFIC_PASSWORD=apple_specific_password
    FASTLANE_USER=apple_id
    FASTLANE_TEAM_ID=team_id
    REAN_APP_IDENTIFIER=kh.gov.rean
    UNIVERSITY_APP_IDENTIFIER=kh.gov.university

    ANDROID_KEYSTORE_PASSWORD=keystore_password
    ANDROID_KEY_ALIAS=kyestore_alias
    ANDROID_KEY_PASSWORD=key_password
    ANDROID_KEYSTORE_PATH=path_to_keystore
    GOOGLE_SERVICE_JSON_KEY_PATH=path_to_google_service_json_key

**Curcious**: udpate fastfile path if you moved the .env-fastlane file to else where
```
